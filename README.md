**Bizim Market** is a modern, real-time shared grocery shopping list application designed for families and households. It allows multiple users to manage a single shopping list simultaneously, view spending statistics, and categorize items efficiently.

Built with **React Native (Expo)** and powered by **Supabase**.

## âœ¨ Key Features

* **Real-Time Synchronization:** Updates made by one family member (adding items, checking off lists) appear instantly on everyone else's device.
* **Family Management:** Easy onboarding with unique invite codes to join family groups.
* **Smart Catalog:** A predefined product catalog with categories (Fruits, Vegetables, Dairy, etc.) for quick adding.
* **Spending Statistics:** Visual pie charts and reports to track monthly shopping habits and category distribution.
* **Premium UI/UX:**
    * **Midnight Gold Theme (Dark Mode):** A sophisticated deep navy and golden yellow palette.
    * **High-Contrast Light Mode:** Optimized visibility with navy accents on a white background.
